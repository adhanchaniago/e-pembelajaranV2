# Kuis Online Codeigniter

Aplikasi Kuis Online Menggunakan CodeIgniter

<h1>Catatan</h1>
<p>Disarankan upgrade PHP ke versi terbaru (7.3 atau lebih tinggi)</p>

<h1>Cara Install</h1>
    Buat database dengan nama <kbd>e-learning</kbd>. Kemudian import database pada folder <kbd>sql</kbd>. Jangan lupa setting lagi file <kbd>config.php</kbd> nya. 
    <br/>

jika sudah mengikuti langkah diatas dengan benar seharusnya aplikasi sudah berjalan dengan baik. Pastikan pada console browser tidak terdapat error.
